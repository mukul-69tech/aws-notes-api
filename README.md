# 🚀 AWS Serverless Notes API

This project is a serverless Notes API built using AWS.

## 🔧 Tech Stack
- AWS Lambda
- API Gateway
- DynamoDB

## 📌 Features
- Create note (POST)
- Get notes (GET)
- Delete note (DELETE)

## 🌐 API Endpoint
https://6phth3bmdc.execute-api.ap-south-1.amazonaws.com/dev/notes
