# 🚀 Notes App (Full Stack Cloud Project)

A full-stack Notes application built using AWS Serverless architecture and modern frontend technologies.

---

## 🌍 Live Demo
👉 https://aws-notes-api.vercel.app/

---

## 🛠 Tech Stack

Frontend:
- React (Vite)
- Tailwind CSS

Backend:
- AWS Lambda
- API Gateway

Database:
- DynamoDB

---

## ✨ Features
- Add Notes
- View Notes
- Delete Notes
